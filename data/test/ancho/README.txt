Ancho Variable Font
==================

This download contains Ancho as both a variable font and static fonts.

Ancho is a variable font with these axes:
  wght

This means all the styles are contained in a single file:
  AnchoGX.ttf

If your app fully supports variable fonts, you can now pick intermediate styles
that arenâ€™t available as static fonts. Not all apps support variable fonts, and
in those cases you can use the static font files for Ancho:
  Ancho-Thin.ttf
  Ancho-Light.ttf
  Ancho-Regular.ttf
  Ancho-Medium.ttf
  Ancho-Bold.ttf
  Ancho-UltraBold.ttf
 
  Ancho-Thin.otf
  Ancho-Light.otf
  Ancho-Regular.otf
  Ancho-Medium.otf
  Ancho-Bold.otf
  Ancho-UltraBold.otf



Get started
-----------

1. Install the font files you want to use

2. Use your app's font picker to view the font family and all the
available styles
